# 선택 정렬
---
- 알고리즘
	1. 앞에서 부터 현재 위치를 옮겨감
	2. 리스트에서 최소값을 찾음
	3. 해당 값을 현재위치와 교환
	4. 현재위치를 한칸 뒤로 옮기고 2~3을 반복함
- 성능
	- 비교 연산 : O(n<sup>2</sup>/2)
		- n-1개와 비교 + n-2개와 비교 ... 1개와 비교
	- 교환 연산 : O(n)
		- n개의 자리에 값을 바꿔 넣어야함
